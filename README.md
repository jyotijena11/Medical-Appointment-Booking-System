# Medical-Appointment-Booking-System

The Medical Appointment Booking System is a full-stack web application designed to
streamline the process of scheduling medical appointments. The project aims to
eliminate long waiting times and administrative hassles by providing a
user-friendly platform for patients to book appointments with doctors and for
doctors to manage their schedules efficiently.

This system serves as a central hub where patients can find doctors by specialty 
location and book their appoinment and they can have access to reschedule it , or 
cancel appointments. Doctors can manage their availability and view patient 
histories, creating a seamless experience for all users.

# Features
This application includes a number of key features to provide a robust and 
efficient experience for both patients and doctors:

# User Authentication: 
Secure registration and login for both patients and doctors, with separate 
dashboards for each user type.

# Doctor and Patient Profiles: 
Doctors can create profiles with their specialty and availability, while patients 
can manage their personal information and view their medical history(if thay are 
login before).

# Search and Filter: 
Patients can easily search for doctors based on their specialization, location, or 
availability.

# Appointment Management:
Users can book appointments, view upcoming and past appointments, and have the 
flexibility to cancel or reschedule their bookings.

# Real-time Availability: 
The system displays doctors' schedules in real-time to prevent double-bookings.

# Automated Notifications: 
The system can send email or SMS reminders to patients and doctors to reduce 
no-shows.

# Technologies Used:
The project is built using a modern full-stack approach with the following 
technologies:

# Frontend:

React.js: A JavaScript library for building the user interface.

HTML & CSS: For structuring and styling the web pages.

JavaScript: The core programming language for the frontend logic.

# Backend:

Node.js/Express.js: A backend framework for building the server and handling API 
endpoints.

MongoDB/PostgreSQL: The database used for storing all users, doctor and 
appointment data.
